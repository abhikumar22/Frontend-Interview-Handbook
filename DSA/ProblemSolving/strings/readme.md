# Strings -

##### Given list of words, get all the words which starts with and are matching with the input string.

```
E.g
list - ["abhi", "abhishek", "hello", "hell", "hi", "abh", "john", "keabh"]
inputStr - "abhi"
output - ["abhi","abhishek"]

E.g
list - ["abhi", "abhishek", "hello", "hell", "hi", "abh", "john", "keabh"]
inputStr - "hell"
output - ["hello","hell"]
```
